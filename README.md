# Portfolio Website

This project is a portfolio website for Sunrise Nguyen, built with React, TypeScript, and Vite.

## How to run

1.  **Build the Docker image:**

    ```bash
    docker build -t nguyen_sunrise_coding_assignment14 .
    ```

2.  **Run the Docker container:**

    ```bash
    docker run -p 5575:5575 --name nguyen_sunrise_coding_assignment14 nguyen_sunrise_coding_assignment14
    ```

    The container name will be `nguyen_sunrise_coding_assignment14`.

3.  **Open the website:**

    Open your browser and go to `http://localhost:5575`.
